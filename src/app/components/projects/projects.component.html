HackWeek 2022 | Microservices

Microfocus (OpenText), Bangalore

Java, Spring Boot, Microservices, Netflix Eureka, gRPC, PostgreSQL, RedisDB

  • Participated in a two-week hackathon event within the company and spearheaded the proof of concept (POC) for refactoring the monolithic model of the Records and Location module into a microservices architecture.
  • Developed two microservices using Spring Boot and Core Java concepts, one for record registration and the other for location registration. Both microservices are registered to the Eureka server for service discovery. Implemented an API Gateway, also registered to the Eureka server, for accessing the microservices.
  • Enabled inter-microservice communication through gRPC calls. Protocol buffers were written and compiled using the protoc compiler to generate Java code models for message passing.
  • Utilized Redis DB as a cache to store frequently accessed data. This approach reduces latency by retrieving data from Redis cache instead of querying the PostgreSQL database for subsequent requests.
